# opentelemetry-php74-instrumentation

[OpenTelemetry](https://opentelemetry.io/) Zero-code instrumentation for PHP 7.4 applications.

> [!WARNING]
> opentelemetry-php74-instrumentation project is currently being released experimentally.

## Components

| Path                                             | Description                                                                          |
| ------------------------------------------------ | ------------------------------------------------------------------------------------ |
| [`ext/`](./ext/)                                 | OpenTelemetry auto-instrumentation extension (`opentelemetry_php74`) providing hooks |
| [`src/Instrumentation/`](./src/Instrumentation/) | auto-instrumentation library using the extension                                     |
| [`examples/`](./examples/)                       | Example Codes                                                                        |
